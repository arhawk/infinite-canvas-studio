Product Overview

Mimi is an infinite-canvas visual organization and presentation tool. Users can create pages, text, sticky notes, images, embedded web pages, code runners, local videos, ranking activities, and shapes, then organize them with outlines, connections, jump buttons, and presentation mode.

Lesson presentationsOrganize multiple Pages into non-linear lessons with Buttons and connections.
Classroom activitiesUse Ranking Box, pens, highlighter, timer, and Page Compare for activities.
Resource navigationKeep web pages, images, videos, attachments, and notes in one canvas.
Online viewingThe teacher-side Host presents the board while student-side Viewers follow or browse freely.

Core Capabilities

  • Infinite canvas with pan, zoom, and overview navigation.
  • Teacher-side modes: Edit for creating and organizing content, Present for maximizing delivery space.
  • Student-side views: Host follows the presenter camera, Viewer allows free panning and zooming.
  • Components including Page, Button, Text, Sticky Note, Image, Iframe, Ranking Box, JS Code Runner, and Local Video.
  • Local JSON save/load and single-file HTML export.

Create Your First Presentation Board

  1. Enter Edit mode.
  2. Click Components in the left toolbar and choose Page.
  3. Add Text, Sticky Note, Image, Iframe, JS Code Runner, or Local Video.
  4. Create a Button and connect it to a target Page or component.
  5. Add key components to the right-side Outline.
  6. Switch to Present and test page navigation, button jumps, and Page Compare.
  7. Export JSON or HTML.

How to use Page

A Page works like a slide. Put related text, images, buttons, and videos on the same Page so they can move together as one presentation unit.

How to use Button

A Button is a one-way jump. For two-way navigation, create one Button on each side and point each Button to the other destination.

Iframe limitation

Some websites block embedding through security policies. If a site cannot display, save it as an attachment or text link instead.

Outline, Connections, And Navigation

Outline

The right-side Outline structures the freeform canvas. Select components and click Add selected, or drag components into the outline. Click an item to center the viewport, double-click to rename it, and drag or use direction controls to adjust order and hierarchy.

Connections

Connections express relationships between components and can also support presentation navigation. Use connections for conceptual relationships and Button for explicit click-to-jump actions.

Overview

Overview is the bottom-right minimap. It shows the current viewport, lets users click to move around, marks selected components, and warns about Pages without connection targets.

Page Compare

In Present mode, select one Page, hold Shift, and click another Page to compare them. The comparison window supports swapping, zooming, fit-to-window, fullscreen, and Esc close.

Save, Export, And Online Sharing

JSONBest for continued editing. Saves components, styles, connections, outline, attachments, drawings, annotations, and viewport state.
HTMLBest for offline delivery or submission. Produces one browser-openable snapshot file.
HostCreate rooms, edit the board, and broadcast board state plus camera position to Viewers.
ViewerFollow Host or browse freely, and save a copy, without editing Host content.
Local file:// HTML exports hide Share because local files cannot reliably use the online room service.

Teaching Helpers And Shortcuts

Helpers

  • Timer: countdowns, timed activities, and presentation pacing. Press Enter in minute or second inputs to finish editing.
  • Binary Calculator: binary calculations and computer science teaching.
  • Overview / Minimap: shows the current viewport, selected component location, page connection status, and supports clicking to move the camera.
  • Page Compare: view two Pages side by side in Present mode. Select one Page, hold Shift, click another Page, and press Esc to close.
  • Background Style: Default/Colorful themes, Blank/Grid/Dot, colors, and opacity.

Mod usually means Ctrl on Windows/Linux and Cmd on macOS.
