这是一份面向最终用户的 documentation：先用“创建第一个项目”建立完整使用路径，再按功能模块解释每个工具的用途。开发者从 0 到 1 启动项目的内容放在末尾附录。

产品概览

Mimi 是一个基于无限画布的可视化内容组织与展示工具。用户可以创建页面、文字、便签、图片、网页嵌入、代码运行器、本地视频、排序活动和图形，并通过目录、连接线、按钮跳转和演示模式组织成可讲解的内容空间。

适合课程演示把多个 Page 组织成非线性课件，用按钮和连接线控制讲解路径。
适合课堂互动使用 Ranking Box、画笔、高亮、计时器和页面对比进行活动。
适合资料导航把网页、图片、视频、附件和说明文本集中到同一画布中。
适合离线提交可导出 JSON 继续编辑，也可导出单文件 HTML 浏览器打开。

创建你的第一个演示画布

  1. 进入 Edit 模式。
  2. 点击左侧 Components，选择 Page 创建第一页。
  3. 继续添加 Text、Sticky Note、Image、Iframe、JS Code Runner 或 Local Video。
  4. 创建 Button 并连接到目标页面或组件，在演示时点击跳转。
  5. 把关键组件加入右侧 Outline，建立目录层级。
  6. 切换到 Present，测试页面导航、按钮跳转和页面对比。
  7. 选择 JSON 或 HTML 导出。
本地打开开发版本 
pnpm install
pnpm dev

访问 http://localhost:3000。

Page 的使用建议

Page 可以理解为一张幻灯片。把相关文字、图片、按钮和视频放在同一个 Page 上，之后移动 Page 时，页面内子组件也能一起移动。

Button 的使用建议

Button 是单向跳转。如果需要 A 和 B 之间双向跳转，请分别在 A 和 B 上创建按钮，并让它们分别指向对方。

界面与模式

主要界面区域

Edit 模式

Edit 用于创建和组织内容。支持单选、多选、框选、移动、缩放、编辑样式、绘图、连接、目录管理和导出。

Present 模式

Present 用于展示。编辑栏会收起，画布空间最大化。可以使用页面边缘浮球、Button、连接线导航入口、页面对比和全屏展示。

在线房间中的 viewer 处于只读观看状态，不能进入 Edit 模式或加载文档覆盖 host 内容。

组件说明

Page基础展示页面，可承载子组件、附件、背景、连接和演示导航。
Button跳转按钮，可连接目标并在 Present 模式下点击跳转。
Text基础文本框，用于标题、说明、问题和注释。
Sticky Note带底色的便签文本，适合提示和讨论记录。
Image上传图片，支持调整大小、连接和随文档保存。
Iframe嵌入网页 URL，可用于展示或交互。
Ranking Box排序活动容器，可把文本项放入后进行排序。
JS Code Runner运行 JavaScript，包含代码、预览和控制台输出区域。
Local Video导入本地视频，使用浏览器播放器控制播放、音量、全屏和速度。
Shapes

Shapes 包括矩形、圆形、菱形和三角形。可以修改文字、字体大小、文字颜色、填充颜色、透明度、边框颜色、边框宽度和层级，并可连接到其他组件。

Iframe 限制

部分网站会因为安全策略禁止被嵌入。如果网页无法显示，可以改为把网址作为附件或文本链接保存。

目录、连接与导航

Outline

右侧 Outline 把自由画布组织成结构化目录。选中组件后点击 Add selected，或拖拽组件加入目录。点击目录项可居中到对应组件，双击可重命名，拖拽或方向按钮可调整层级和顺序。

连接线

连接线用于表达组件关系，也可以作为演示导航路径。建议用连接线表达内容关系，用 Button 表达明确的观众点击跳转。

Overview

Overview 是右下角小地图。它显示当前视口位置、支持点击移动视角、显示选中组件定位提示，并能提示没有连接对象的 Page。

Page Compare

在 Present 模式下，选中一个 Page 后按住 Shift 点击另一个 Page，即可打开页面对比。对比窗口支持交换位置、缩放、适配窗口、全屏和 Esc 关闭。

保存、导出与在线分享

JSON适合继续编辑。保存组件、样式、连接、目录、附件、绘图、注释和视口状态。
HTML适合离线展示或提交。生成一个浏览器可打开的单文件快照。
Load加载 JSON 会替换当前画布，加载完成后成为新的 undo/redo 基线。
Share创建四位房间链接和二维码，可设置可选密码。

在线房间

Host 创建房间后可以分享链接。Viewer 打开链接后可以跟随 host 视角，也可以切换自由观看视角。Viewer 可以保存副本，但不能编辑 host 的画布。

本地 file:// 打开的 HTML 导出版会隐藏 Share 按钮，因为本地文件不能稳定使用在线房间服务。

辅助工具与快捷键

课堂辅助

  • Timer：用于倒计时、课堂活动计时或展示节奏控制。

开发者运行附录

这部分不是最终用户手册的主体，只是给需要从 0 到 1 启动项目的人一个最小路径。

环境要求

  • Node.js ^20.19.0 或 >=22.12.0
  • pnpm 10.23.0

本地运行

pnpm install
pnpm dev

房间服务

pnpm server

构建与测试

pnpm build
pnpm test:unit
pnpm test:e2e
pnpm test

二次开发提醒

  • 组件放在 src/component/，行为放在 src/plugins/。
  • 组件需要实现创建、序列化和恢复。
  • 可撤销修改应发出 node:change:start 和 node:changed。
  • 新组件至少要通过 undo/redo 或 document load 的 roundtrip 测试。
  • 普通前端功能不要修改 server/，除非明确需要后端变化。
