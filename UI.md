# UI 设计规范

> 参考风格：极简

---

## 色彩

| Token | Hex | 用途 |
|---|---|---|
| `--primary` | `#B85042` | 强调色、激活态、主按钮 |
| `--bg` | `#FAF6F0` | 页面和画布背景 |
| `--surface` | `#FFFFFF` | 面板、卡片、输入框 |
| `--text` | `#2D2D2D` | 主要文字 |
| `--text-secondary` | `#8B8B8B` | 辅助文字、占位符 |
| `--border` | `#E5E5E5` | 边框、分隔线 |
| `--danger` | `#D94040` | 清除、删除 |
| `--grid` | `#E8E0D8` | 画布网格线 |

**原则：** 保持表面干净，避免多余的背景色或渐变，让留白来承担设计功能。

---

## 字体

- **面板标题：** 13px，加粗，全大写（如 `OVERVIEW`、`OUTLINE`）
- **表单标签：** 14px，半粗
- **正文：** 14px，常规
- **小号文字：** 12px，次要颜色

保持简洁——一种字体，仅通过字重和字号区分层级。

---

## 间距（仅参考）

基准单元：**4px**，使用倍数：4、8、12、16、24。

- 组件内边距：**16px**
- 元素间距：**8px**
- 区块分隔：**24px**

---

## 圆角与阴影（仅参考）

- 面板：`border-radius: 16px`
- 按钮和输入框：`border-radius: 8px`
- 工具栏图标：`border-radius: 50%`（圆形）
- 面板阴影：`0 2px 8px rgba(0,0,0,0.08)` — 保持轻微

---

## 组件规则

### 关闭按钮

✅ 使用 **× 图标**，放在右上角

❌ 不要使用文字按钮（如 "Close"）

### 折叠

✅ 折叠箭头 **˅/˄** 嵌入面板自身标题栏

❌ 不要把折叠控件放在外部工具栏

### 工具栏

✅ 每类工具一个图标，子工具用**下拉菜单**选择（如画笔类型）

❌ 不要将所有子工具并排显示在工具栏

### 按钮（仅参考）

- **主按钮：** `--primary` 背景 + 白色文字，8px 圆角
- **次要按钮：** 白色背景 + `--border` 边框 + 深色文字
- 最小触摸目标：**44px × 44px**

### 面板定位

| 面板 | 位置 |
|---|---|
| 工具栏 | 顶部居中，固定 |
| 小地图 | 右下角，可折叠 |
| 大纲 | 右侧，可折叠 |
| 编辑器 | 居中浮层 |
| 计算器 | 浮动可拖拽 |

提交涉及 UI 的 PR 前请确认：

- [ ] 关闭按钮用 × 图标
- [ ] 折叠箭头在面板标题栏内
- [ ] 颜色使用规范中的 token
- [ ] 所有可点击元素有悬停态
- [ ] 无视觉杂乱——拿不准就删掉
